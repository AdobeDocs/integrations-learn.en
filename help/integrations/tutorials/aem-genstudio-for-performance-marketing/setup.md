# Set up GenStudio for Performance Marketing and AEM Assets

[â€¹ Back to integration overview](./overview.md)

To begin using Adobe GenStudio with AEM Assets as a Cloud Service and GenStudio for Performance Marketing, ensure the necessary products are provisioned, enabled, and configured. Specifically, you need access to:

* AEM Assets as a Cloud Service
* AEM Assets Content Hub
* GenStudio for Performance Marketing

There are no additional setups required to integrate these products beyond ensuring they are available, and set up and accessible to your user.

## Setup guides

If these Adobe products are not yet set up and enabled, consult the following guides for detailed configuration steps:
